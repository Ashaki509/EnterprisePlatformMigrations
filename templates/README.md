# 📁 Templates

Production-ready planning templates for enterprise platform migrations. Built from hands-on program delivery across $250M+ portfolios — not generic frameworks.

---

## 📄 Files in This Folder

### 🔴 [risk-assessment-matrix.xlsx](risk-assessment-matrix.xlsx)
Probability × Impact scoring matrix for identifying and prioritizing migration risks.
- 5×5 color-coded heat map (Low → Critical)
- 10 pre-populated migration risks with mitigation and contingency strategies
- Auto-calculated risk scores with CRITICAL / HIGH / MEDIUM / LOW ratings
- Status tracking with dropdown validation

---

### 🔵 [raci-migration-template.xlsx](raci-migration-template.xlsx)
Responsibility assignment matrix for cross-functional migration teams.
- 25 activities mapped across all 5 migration phases
- 13 roles from Executive Sponsor through End User Rep
- Color-coded R / A / C / I cells for instant readability
- Covers Discovery → Planning → Pilot → Execution → Stabilization

---

### 🟢 [stakeholder-communication-plan.xlsx](stakeholder-communication-plan.xlsx)
End-to-end communication framework for keeping stakeholders aligned throughout the migration.
- Stakeholder register with Interest, Influence, and engagement strategy per group
- 11 communication events from kick-off through executive close-out
- Trigger timing, owner, channel, format, and success metric for every event

---

### 🟠 [milestone-tracker.xlsx](milestone-tracker.xlsx)
20-week Gantt-style timeline with live summary dashboard.
- 27 tasks across 5 phases with visual Gantt bars
- Live dashboard: On Track / At Risk / Delayed / Complete / % Done
- Priority and Status dropdowns; Actual vs. Planned date tracking
- Structured for direct import into Jira (CSV) or Smartsheet

---

## 🚀 How to Use

1. Download the template(s) relevant to your current phase
2. Replace placeholder owners and dates with your team's actuals
3. Use the Risk Matrix first — it will inform your RACI and Comms Plan
4. Import the Milestone Tracker into Jira or Smartsheet if preferred
